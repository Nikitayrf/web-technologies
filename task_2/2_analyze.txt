Сайт свободной общедоступной многоязычной энциклопедии "Википедия".
Сайт состоит из статей. Каждый может добавить и вносить изменения.

Устройство заглавной страницы.

1. Шапка страницы занимает всю ширину страницы и высоту ~ 80рх.
    Включает в себя горизонтальные подменю:
    > Личный кабинет и связанные вкладки.
    > Поиск по сайту
    > Меню:
        >> Читать
        >> Просмотр кода
        >> История
    > Меню:
        >> Заглавная
        >> Обсуждение

2. Под шапкой страница разделена на две колонки:
    Узкая левая. Навигационная панель шириной ~ 200px
    Правая широкая. Основной контент ~ 800px
 
 2.1 Навигационная панель состоит из подменя:
    >> Навигация по текущей странице
    >> Участие
    >> Инструменты
    >> Печать/экспорт
    >> В других проектах
    >> На других языках

 2.2 Основной контект в свою очередь разделен:
        Начинается с меню шапки шириной ~ 800px
    2.2.1 Шапка основного контента:
        >> Порталы
        >> Справка
        ** Создать статью

        Разделение основного контента на 2 равные колоки статей шириной ~ 400px
    2.2.2 Левая колонка статей:
        >> Избранная статья
        >> Хорошая статья
        >> Последний избранный список
        >> Изображение дня
    2.2.3 Правая колонка статей:
        >> Из новых материалов
        >> Текщие события
        >> В этот день
        >> Совместная работа недели

        Под статьями расположен подвал основного контента шириной ~ 800px
    2.2.4 Подвал основного контекта:
        >> Ссылки на связанные проекты

3. Футер заглавной страницы занимает размещается после основного контента.
    Футер сотстоит:
    >> Правовая информация
    >> Меню связи
    >> Статистика
